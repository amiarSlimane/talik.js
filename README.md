## ⭐️ The Library uses

- Webpack 5
- Babel 7
- Automatic Types file generation (index.d.ts)
- UMD exports.
- Jest unit testing
- Daily [dependabot](https://dependabot.com) dependency updates

## 📦 Install

```
npm install talik
```

## 🚀 Usage
 

```
import Talik from 'talik.js';
const TalikInstance = new Talik();
talikInstance.init(); 
...
```
